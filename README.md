## Clone

```
git clone https://github.com/phpaulohenrique/focal-point-back-end.git
npm install
```

<h5>Docker</h5>

```
 docker compose up -d
```

<h5>Fill the .env</h5>

```
DATABASE_URL=""
PORT=3333
FRONTEND_URL="http://localhost:3000"
```

<h5>Start server</h5>

```
  //in root
  npm run dev
```
